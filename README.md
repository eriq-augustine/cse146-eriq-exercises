# 146 Eriq Exercises

These are exercises that go along with Eriq's CSE146 lecture.

For these exercises, take a look at the PRs and decide what comments (if any) you would give for the PR.

All the code here has came from real PRs,
but has been stripped down, converted to Python, and modified to highlight the specific issues.
Note that because this code has been stripped down from real projects (which may have been quite large),
you may have to use your imagination at some points.
